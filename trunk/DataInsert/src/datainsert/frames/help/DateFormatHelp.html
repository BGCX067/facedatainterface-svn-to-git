Formatação de datas

Forma com que as datas são apresentadas ao usuário. Ex: 12/7/2010, 12 de julho de 2010, 12-jul-2010, 7/12/2010, 12-7-2010, etc.

É importante para a importação dos dados do arquivo que a formatação de datas encontradas no arquivo coincida com a encontrada na base de dados, caso contrário há incompatibilidade no momento da inserção. Caso o formato da data esteja diferente do encontrado na base de dados, é crucial que o usuário escolha o formato de data na caixa de escolha formatar data (abaixo dos dispositivos a inserir).

Os formatos de data encontrados no programa são:

  • DD/MM/YYYY;
  • MM/DD/YYYY;
  • YYYY/MM/DD;
  • DD/MM/YY;
  • MM/DD/YY;
  • YY/MM/DD.

Sendo:

  • MM : Mês, numérico;
  • DD : Dia, numérico;
  • YYYY : Ano, numérico, quatro digitos;
  • YY : Ano, numérico, dois digitos.
