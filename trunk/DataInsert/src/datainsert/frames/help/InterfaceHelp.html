Componentes de interface gráfica

Esta seção apresentará ao usuário as seções e operações da interface gráfica do aplicativo.

  • Seleção de tabela de dados:
    • - Tabela de dados: Tabela em que se quer inserir os dados
    • - Guia de dispositivos: Identificação dos dispositivos relacionados à tabela.

Figura 1. Seleção da tabela de dados.

  • Inserção de dados:
    • - Dados do arquivo importado;
    • - Dispositivos do arquivo: Dispositivos que geraram os dados encontrados no arquivo;
    • - Dispositivos a inserir: Dispositivos selecionados para a inserção dos seus respectivos dados;
    • - Formatar data: Formato de data encontrado no arquivo de dados.

Figura 2. Seleção dos parâmetros de inserção.

  • Conexão: (Conexão > Abrir Conexão)
    • - IP: IP (rede local) da máquina em que se encontra a base de dados;
    • - Porta: Porta destinada ao serviço do sistema gerenciador do banco de dados (no caso do PostgreSQL a porta default é a 5432);
    • - Usuário: Nome do usuário da base de dados;
    • - Senha: Senha de acesso do usuário;
    • - Base de dados: Nome da base de dados a ser utilizada.

Figura 3. Seleção dos parâmetros de conexão.

  • Definir Separadores: (Arquivo > Definir Separadores)
    • - Máscara: Caracter que irá separar os dados do arquivo de forma consistente;
    • - Adicionar: Adiciona a máscara a uma lista de máscaras;

Figura 4. Definição de separadores de dados.

  • Remover dados: (Dados > Remover)
    • - De: Data de início (Caso esteja vazio, serão removidos os dados da data estipulada no outro ComboBox);
    • - Até: Data de fim(Caso esteja vazio, serão removidos os dados da data estipulada no outro ComboBox);
    • - Valor do dado: Serão removidos todos os dados dentro dos parâmetros escolhidos caso esteja vazio;
    • - Dispositivos: Dispositivos encontrados na tabela de dados;
    • - A remover: Dispositivos cujos dados serão removidos.

Figura 5. Remoção de dados.
