Separação do arquivo de dados

Define ao aplicativo a forma do mesmo interpretar as informações do arquivo de dados.

Para a importação do arquivo de dados é necessário, por meio de algum artificio, que o aplicativo interprete o arquivo coletando os dados relevantes. Para isto, o usuário deverá definir como os dados estão separados no arquivo. Por exemplo, em um arquivo podemos ter dados separados por espaçamento, tabulação, ponto, ponto e virgula, virgula, etc. Logo, o usuário é encarregado de definir este parâmetro.

O usuário pode encontrar esta operação em Arquivo > Definir Separadores.
