Tabela de dados

A tabela de dados armazenará os dados escolhidos pelo usuário. Tal tabela representa os dados gerados por um Nó ou Anel contendo os dados e informações sobre os mesmos tais como o dispositivo rque o gerou, data e hora de coleta e qualidade.

O usuário deverá definir a tabela de dados referente ao Nó ou Anel de que ele irá armazenar os dados coletados. A seleção de tabelas de dados se encontra na sessão "Seleção da tabela de dados". Esta sessão também Possui a guia de dispositivos para facilitar a inserção dos dados.
