Seleção de dispositivos

Para garantir a consistência para com a base de dados, o usuário deve identificar a qual dispositivos os dados pertencem.

A seleção de dispositivos caracterizará a inserção do usuário. O mesmo deverá selecionar os dispositivos dentro da seção dispositivos do arquivo e pressionar o botão ">>" para enviar tais dispositivos para lista de dispositivos a inserir. O rótulo dos dispositivos na seção dispositivos do arquivo são os mesmos encontrados no arquivo de importação. Ao selecionar os dispositivos, o usuário deverá escolher o canal referente ao mesmo a partir da caixa de seleção ao lado do rótulo do dispositivo, tudo isto dentro da seção dispositivos a inserir.

Na sessão Seleção de tabela de dados será apresentado um guia de dispositivos para que o usuário possa identificar os canais de cada dispositivo.
