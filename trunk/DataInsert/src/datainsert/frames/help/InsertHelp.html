Inserção

Esta seção apresentará a sequência de operações a serem executadas para a inserção dos dados na tabela de dados

Para efetuar uma inserção na tabela de dados deve-se, primeiramente, selecionar as separações utilizadas no arquivo de dados para que o aplicativo reconheça de forma consistente a formatação do arquivo de dados gerado pelos sensores do experimento. Deve-se, também, conectar-se à base de dados a que se queira importa-los.

Após este procedimento, deve-se selecionar o arquivo a ser importado selecionando Arquivo > Abrir ou o botão Arquivo. Serão apresentados os dados na seção "Inserção de dados / Dados do arquivo importado". Deve-se então, selecionar os dispositivos referentes a cada nó no arquivo importado utilizando a lista "Inserção de dados / Dispositivos a inserir" e selecionar a tabela de dados referente ao nó escolhido na seção "Seleção da tabela de dados / Tabela de dados".

O usuário deverá verificar se a formatação de data encontrada na caixa de opção "Formatar data" condiz com a do arquivo importado. Caso não esteja condizente, o usuário deverá selecionar o formato de data ideal.

Pode-se então, relacionar os dispositivos na seção "Inserção de dados / Dispositivos à inserir" aos seus respectivos canais utilizando a seção "Seleção da tabela de dados / Guia de dispositivos" e logo após, efetuar a operação de inserção pelo botão "Inserir Dados".
