Remoção

A remoção é uma ferrameta implementada para caso haja inserção incorreta ou dados incorretos na base de dados.

A base de dados relacionada aos dados angariados pelo experimento possui uma grande quantidade de registros e, portanto, inviabiliza-se a manipulação atômica das tuplas. Assim, a remoção de dados foi implementada de tal forma que esta utilize um filtro de informações pré-determinado pelo usuário final a fim de remover do armazenamento persistente um conjunto de registros abrangido por tal filtro.

Os elementos do filtro que o usuário final pode determinar são os respectivos:

  • Intervalo de data
    • Se o usuário final desejar excluir os dados de apenas uma data, o intervalo será da data desejada até a mesma.
  • Valor do dado a ser removido
  • Dispositivos

Antes de iniciar a remoção o usuário deverá estar conectado ao banco de dados e deverá ter selecionado uma tabela de dados para aplicar a remoção a mesma.

Notas:

  • Para remover todos os dados deve-se utilizar o caracter '*' em algum dos campos de dados;
  • Para remover um dado específico deve-se utilizar apenas um campo de dados;
  • Para remover um intervalo de dados específico deve-se utilizar os dois campos de dados.
