Base de dados

Estrutura modelada para garantir coerencia entre os dados e informações do experimento. Possui também como objetivo o armazenamento destes em memória não-volátil.

A base de dados utilizada no experimento é grande e complexa. Porém, o escopo deste aplicativo engloba apenas os dados coletados pelos sensores e atuadores dos ensaios restringindo as informações necessárias ao usuário. Utilizar o aplicativo necessitará de conhecimento sobre 1) qual a identificação do nó gerador dos dados 2) os dispositivos instalados neste e suas identificações 3) a tabela de dados referente ao nó.

A fim de facilitar o uso da interface, criou-se uma guia para que o usuário identifique os canais em que os dispositivos de um determinado nó estão instalados. Esta guia é encontrada na seção "Guia de dispositivos".
